# Color Contrast Calculator

This project is a simple and interactive **color contrast checker** that helps users test text and background color combinations for accessibility. It displays the contrast ratio, WCAG pass/fail status, and allows users to edit preview text in real time.

## 🔗 Live Demo
You can view the live version of the project here:

https://students.washington.edu/ctoun/calculator.html

## 📁 Project Files
- **calculator.html** — Main webpage structure  
- **calculator.css** — Styling for layout, colors, and UI  
- **calculator.js** — Logic for calculating contrast and updating the UI  

## 🎯 Features
- Real-time color picking  
- Editable preview text  
- Automatic contrast ratio calculation  
- WCAG accessibility feedback  
- Clean, modern UI
